Search Engines

DogpileReturns the top search results from multiple other search engines.
AskAllows common questions to be entered as a search key. Provides smart search for products and weather.
GoogleProvides general keyword and phrase searching. Provides definition search by typing "definition" and a term.
HotBotProvides general keyword search, also the toolbar allows you to search your local computer.
BingGeneral keyword search, also allows searches to be limited to specific file types.
YahooProvides news search and tabbed categories to help locate information.
